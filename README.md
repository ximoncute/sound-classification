# Sound-classification
🎧🤖 Sound Classification with AI – When Machines Learn to Listen

🌐 Web demo: https://ximoncute.github.io/he_thong_an_ninh_thong_minh/

🚀 Sound-classification là một dự án AI / Machine Learning / Deep Learning tập trung vào việc dạy cho máy tính biết nghe – hiểu – và phân loại âm thanh giống như con người.

🔊 Âm thanh không chỉ là sóng – đó là thông tin. 

🧠 AI không chỉ tính toán – nó học cách cảm nhận.

🌟 Giới thiệu tổng quan
Trong kỷ nguyên Artificial Intelligence, việc xử lý và hiểu âm thanh đóng vai trò quan trọng trong các hệ thống hiện đại như: 

🎙️ Trợ lý ảo, 

🏙️ giám sát môi trường,

🤖 robot thông minh, 

📡 IoT & Edge AI,

🔐 an ninh – nhận diện sự kiện âm thanh.

👉 Sound-classification được xây dựng nhằm mô phỏng khả năng “nghe và phân biệt” của con người bằng các mô hình AI hiện đại.

🧠✨ AI làm được gì trong dự án này?

✔️ Nghe file âm thanh (.wav / .mp3)

✔️ Chuyển âm thanh → dữ liệu số

✔️ Trích xuất đặc trưng thông minh

✔️ Huấn luyện mô hình học máy / học sâu

✔️ Dự đoán & phân loại âm thanh mới

📌 Từ sóng âm → dữ liệu → tri thức → quyết định.

🧩 Kiến trúc AI tổng thể

Âm thanh 🎵 → Tiền xử lý 🔧 → Trích xuất đặc trưng 🧬 → Mô hình AI 🤖 → Phân loại / Dự đoán 🎯

🔥 Đây là pipeline chuẩn của một hệ thống AI xử lý âm thanh chuyên nghiệp.

🧬 Trí tuệ nhân tạo được sử dụng
🧠 Machine Learning: Support Vector Machine (SVM), Random Forest, K-Nearest Neighbors (KNN).

🤖 Deep Learning: Convolutional Neural Network (CNN), Neural Network học trên Spectrogram / MFCC.

💡 CNN giúp AI “nhìn thấy âm thanh” dưới dạng ảnh phổ tần số, từ đó học được những pattern mà tai người khó nhận ra.

🎼 Xử lý âm thanh thông minh
🔹 Chuyển đổi âm thanh sang tín hiệu số
🔹 Chuẩn hóa & lọc nhiễu
🔹 Chuyển đổi sang miền tần số

📌 Đặc trưng AI sử dụng: 🎚️ MFCC, 🌈 Mel-Spectrogram, 📈 Spectrogram.

👉 Đây chính là “DNA của âm thanh” để AI học.

📁 Cấu trúc dự án

Sound-classification gồm các thành phần chính: data (dữ liệu âm thanh), preprocessing (tiền xử lý), feature_extraction (trích xuất đặc trưng), models (mô hình AI), training (huấn luyện), evaluation (đánh giá), demo (dự đoán thử) và main.py là chương trình chính.

📦 Thiết kế rõ ràng – dễ mở rộng – chuẩn AI project.

🛠️ Công nghệ & thư viện
Ngôn ngữ: Python 🐍

Thư viện: Librosa (xử lý âm thanh), NumPy / SciPy, Scikit-Learn, TensorFlow / PyTorch, Matplotlib.

🚀 Hướng dẫn chạy nhanh

Cài đặt: pip install -r requirements.txt

Huấn luyện: python train.py

Dự đoán: python predict.py --audio test.wav

🧠 AI trả về loại âm thanh và độ tin cậy (%).

📊 Đánh giá mô hình AI
Accuracy, Precision, Recall, F1-Score, Confusion Matrix – giúp kiểm tra AI nghe đúng hay chưa.
